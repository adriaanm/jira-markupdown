Byte z according to specification should weakly conform to Char (3.5.3). But this code doesn't compile.
Yes, we can call this ticket a reopening of SI-3143.
Spec now says the following, closing as fixed.

Byte <:w Short
Short <:w Int
Char <:w Int 
Int <:w Long 
Long <:w Float
