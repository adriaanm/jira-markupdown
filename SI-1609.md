In next code:

{code}
object T extends Application {
  class A
  val b: A = null
  print(b match {
    case _: A => true
    case _ => false
  })
}
{code}
We see compiler error: unreachable code.
But code:

{code}
object T extends Application {
  class A
  val b: A = null
  print(b match {
    case _: A => true
  })
}
{code}
produce match error: null.
