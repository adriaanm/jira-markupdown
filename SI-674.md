{code}
def curry : T1 => ... => Tn => R = 
     { (x1: T1) => .... (xn: Tn) => apply(x1, ..., xn) }

Of course, it ought to be possible to rewrite it as 
{code}
def curry : T1 => ... => Tn => R = 
     { (x1: T1) => ( (x2: T2, ..., xn: Tn) => this.apply(x1, ..., xn)).curry }
{code}
