The bug is trivial... ConstructingParser.fromFile uses io.Source.fromFile, without bothering to close the file.

https://github.com/scala/scala/blob/master/src/library/scala/xml/parsing/ConstructingParser.scala

That in turn can cause an exception if one calls ConstructingParsers.fromFile enough times to fill up the maximum open files limit.

Example:
