The {{concurrent.Map}} trait should override {{getOrElseUpdate}} as follows:

{code}
override def getOrElseUpdate(key: A, op: => B): B = get(key) match {
  case Some(v) => v
  case None =>
    val v = op
    putIfAbsent(key, v).getOrElse(v)
}
{code}
